## Cloud Mastery Series

**Location:** Academic Hall, 5th Floor, FPT University Ho Chi Minh City Campus

---

### Overview

**Cloud Mastery Series** is a series of in-depth technical events organized by First Cloud AI Journey at the FPTU HCM Campus. The series consists of multiple workshops and seminars focusing on two main pillars: **DevOps** and **Generative AI**. It provides both fundamental and advanced practical knowledge, helping interns gain hands-on experience with industry-standard technologies widely used on AWS.

---

### Session Details

#### Track 1 — DevOps

| Session | Topic | Description |
|------|--------|-------|
| Cloud Mastery #2 | **Infrastructure as Code (IaC)** | Introduction to IaC, using Terraform and AWS CloudFormation to manage infrastructure as code, ensuring consistency and reproducibility in deployments. |
| Workshop — Terraform | **Advanced Terraform Setup** | Hands-on guide for advanced Terraform configuration: modules, state management, workspaces; including practical demo videos. |
| Workshop — LocalStack | **LocalStack Demo** | Demonstration of using LocalStack to simulate the AWS environment locally, enabling cost-free cloud development and testing. |
| Workshop — K8s | **Kubernetes (Part 1)** | Introduction to the Kubernetes platform: cluster architecture, Pods, Deployments, Services; and how to deploy containers on K8s. |
| Workshop #6 | **Elixir in DevOps** | Exploring the Elixir language and its applications in DevOps automation tasks. |

#### Track 2 — Generative AI

| Session | Topic | Description |
|------|--------|-------|
| GenAI — Prompt Engineering | **Automated Prompt Engineering: Enhancing LLM Output Quality** | Optimization techniques for Large Language Models (LLMs), focusing on structured prompt writing to improve output quality — Speaker: Nguyen Tuan Thinh. |
| GenAI — AIoT | **AIoT Projects** | Projects combining AI and the Internet of Things (IoT) on the AWS platform, focusing on real-world applications in monitoring and automation. |
| GenAI — Strands Agent | **Strands Agent** | Introduction to the Strands Agent model in the AI ecosystem; building and deploying AI Agents on AWS. |

### Key Takeaways

By participating in the Cloud Mastery Series, I have acquired valuable knowledge directly applicable to my internship:

- **Infrastructure as Code & Terraform:** Gained a deeper understanding of infrastructure management via code—knowledge that directly supports writing Terraform modules for the SmartHire AI project (S3, SQS, Lambda, DynamoDB).

- **LocalStack:** Learned how to simulate AWS environments locally to develop and debug faster, saving costs and reducing deployment wait times while developing the CV processing pipeline.

- **Kubernetes:** Although the SmartHire AI project utilizes a Serverless architecture, basic knowledge of K8s helps me understand container orchestration—another popular deployment model on AWS (EKS).

- **Prompt Engineering:** This section is highly relevant to the SmartHire AI project, where Amazon Bedrock is used for CV analysis. Structured prompt techniques significantly improve the quality of data extraction results from CVs.

- **AI Agent (Strands Agent):** Expanded my perspective on the trend of building automated AI systems on AWS, providing foundational knowledge for future AI projects.
